# Google Front Page Clone

This is a simple clone of the **Google home page**, built using **HTML** and **Tailwind CSS**.  
I created this project to practice front-end design, layout, and responsive styling.

## Features
- Clean and minimalist design  
- Centered Google-style search bar  
- Responsive layout using Tailwind CSS  
- Hover and focus effects for interactivity

## Technologies Used
- HTML5  
- Tailwind CSS
